Recursos computacionais

Neste curso usaremos a linguagem de programação Python e os pacotes de computação científica NumPy e SciPy. Usaremos também o pacote Matplotlib para desenhar gráficos. Para interagir com o código usaremos cadernos Jupyter.

Para usar estes recursos, o aluno tem duas opções:

  • Instalar em sua máquina a distribuição científica de Python Anaconda. Instruções para usuários Windows:
    1. Baixe Anaconda (versão Python 3) neste link.
    2. Clique no executável e siga as instruções de instalação.
    3. Clique no ícone do Jupyter Notebook no menu Iniciar. Você verá em seu navegador de internet o conteúdo de uma pasta em seu computador, provavelmente "C:\Users\SEU_NOME\Documents".
    4. Clique em "New" e então em "Python [Root]", sob "Notebooks".
    5. Na caixa à direita de "In [1]:", digite "2 + 2" e aperte Shift+Enter. Se aparecer 4, você está pronto pra começar!
  • Trabalhar na nuvem, usando o Sage Math Cloud. Instruções:
    1. Clique no link e crie uma conta ou conecte uma já existente (do Facebook ou do Google, por exemplo).
    2. Clique no botão "Create new project...", escolha um título (por exemplo, "Teste"), e clique em "Create project without upgrades".
    3. Um projeto com o título que você deu deve aparecer. Clique nele e espere a mensagem "Starting..." desaparecer.
    4. Clique em "New", em cima, à esquerda. Clique na opção "Jupyter Notebook". Espere o site carregar.
    5. Na caixa à direita de "In [1]:", digite "2 + 2" e aperte Shift+Enter. Se aparecer 4, você está pronto pra começar!
